# typescript-playground

(21-Mar-2025) A sandbox for compiling and running your TypeScript code.

## Installation

* [NodeJS](https://nodejs.org/en/).

```bash
npm i -g yarn
```

## Compile

```bash
yarn compile
```

## Usage

```bash
yarn start
```
